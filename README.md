# Бот напоминаний о днях рождения (Telegram)

Бот для коллег: регистрация (имя, фамилия, дата ДР, команда), напоминания за 7 / 6 / 4 / 2 дня, создание отдельного поздравительного чата (сценарий A2), поздравление именинника в день ДР.

## Что нужно заранее

- Windows с интернетом
- [Python 3.11+](https://www.python.org/downloads/) — при установке отметьте **«Add Python to PATH»**
- Аккаунт Telegram

---

## Шаг 1. Создать бота в Telegram

1. Откройте [@BotFather](https://t.me/BotFather).
2. Отправьте `/newbot`.
3. Введите **имя** (как видят пользователи, например: `Дни рождения команды`).
4. Введите **username** латиницей, должен заканчиваться на `bot`, например: `myteam_birthday_bot`.
5. BotFather пришлёт **токен** вида `7123456789:AAH...` — сохраните, это секрет.

По желанию в BotFather:

- `/setdescription` — кратко, зачем бот
- `/setcommands` — вставьте:

```
start - Регистрация
profile - Изменить данные
help - Справка
```

---

## Шаг 2. Узнать свой Telegram ID (для тестов)

1. Откройте [@userinfobot](https://t.me/userinfobot).
2. Нажмите Start — бот пришлёт ваш **Id** (число).
3. Запомните его для файла `.env`.

## Шаг 3. Настроить проект на компьютере

Откройте PowerShell и выполните:

```powershell
cd "c:\Users\Lenovo\Desktop\ИИ. Обучение\Cursor\telegram-birthday-bot"
```

Создайте виртуальное окружение и установите зависимости:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Скопируйте настройки:

```powershell
copy .env.example .env
```

Откройте файл `.env` в блокноте и заполните:

```env
BOT_TOKEN=вставьте_токен_от_BotFather
TIMEZONE=Europe/Moscow
REMINDER_HOUR=9
ADMIN_TELEGRAM_IDS=ваш_telegram_id
```

Сохраните файл.

---

## Шаг 4. Запустить бота

В том же окне PowerShell (с активированным `.venv`):

```powershell
python main.py
```

Должно появиться что-то вроде: `Scheduler: daily at 9:00 ... Bot starting...`

Окно **не закрывайте** — пока оно открыто, бот работает. Для постоянной работы позже можно VPS или планировщик Windows (см. ниже).

## Шаг 5. Проверить в Telegram

1. Найдите бота по username из BotFather.
2. **Start** → «Начать регистрацию» → имя, фамилия, дата `ДД.ММ`, команда.
3. Попросите 1–2 коллег сделать то же (или второй аккаунт).
4. Для **теста напоминаний** (только ваш ID в `ADMIN_TELEGRAM_IDS`):

   - Временно в регистрации укажите дату рождения **ровно через 7 дней** от сегодня (посчитайте в календаре).
   - В личке боту отправьте: `/test_reminders`
   - Коллегам (кроме этого «именинника») должно прийти напоминание за неделю.

## Шаг 6. Разослать ссылку коллегам

Текст для чата команды:

```
Коллеги, подключитесь к боту дней рождения — напоминания только в личку, без спама в общий чат.

1) Откройте: https://t.me/ВАШ_BOT_USERNAME
2) Start → укажите имя, фамилию, дату ДР (ДД.ММ) и команду

Без регистрации напоминания не приходят.
```

Подставьте username бота вместо `ВАШ_BOT_USERNAME`.

## Как пользоваться: поздравительный чат (A2)

1. За 7 дней (или по напоминаниям −6 / −4 / −2) кому-то приходит кнопка **«Создать поздравительный чат»**.
2. Инициатор нажимает → получает инструкцию.
3. Создаёт **новую группу**, добавляет бота, делает **админом** (право: приглашать по ссылке).
4. В группе пишет `/done`.
5. Бот рассылает ссылку **всем зарегистрированным**, кроме именинника.
6. У инициатора **12 часов** на шаги 3–4, иначе может нажать другой.

**Важно:** именинник не должен получать ссылку и не должен быть добавлен в группу вручную.

---

## Расписание напоминаний

| Когда | Кому | Что |
|--------|------|-----|
| ДР − 7 дней | все зарегистрированные, кроме именинника | напоминание + кнопка создать чат |
| ДР − 6, −4, −2 | то же, если чата ещё нет | мягкие напоминания (через день) |
| после `/done` | все, кроме именинника | ссылка на чат |
| ДР − 2 дня | зарегистрированные, не вступившие в чат | «вы не вступили» |
| день ДР | только именинник | поздравление (ротация текстов) |

Ежедневная проверка в **REMINDER_HOUR** (по умолчанию 09:00, часовой пояс **TIMEZONE**).

## Частые проблемы

**«Укажите BOT_TOKEN»** — не заполнен `.env` или неверный токен.

**Бот не отвечает** — проверьте, что `python main.py` запущен и нет ошибки в консоли.

**Напоминания не приходят** — пользователь не нажал Start / не завершил регистрацию; дата ДР не попадает на «сегодня + 7 дней»; бот не был запущен в 09:00 (используйте `/test_reminders`).

**`/done` не работает** — команда только в **группе**; бот должен быть **админом** с правом invite link.

**Коллега не получает ЛС** — он не нажал Start у бота или заблокировал бота.

## Работа 24/7 (без вашего ПК) — Render бесплатно

Пошаговая инструкция: **[RENDER_DEPLOY.md](RENDER_DEPLOY.md)**

После деплоя **не запускайте** бота на ПК с тем же токеном.

На бесплатном Render при **пересборке** база SQLite может обнулиться — см. RENDER_DEPLOY.md.
